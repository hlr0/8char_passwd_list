# 8char_passwd_list

------------------------------------------------
### Description
------------------------------------------------
Generates password lists in number sequence of 48mb "PER" txt files for cracking wifi passwords

* uses lowercase
* uses uppercase
* uses numbers
* uses specialchars

------------------------------------------------
### Sequence of letters in files are: 
------------------------------------------------

* aaaaaaaa
* aaaaaaab
* aaaaaaac
* AABBbbcc
* AABBb123
* ABcd&*)#
